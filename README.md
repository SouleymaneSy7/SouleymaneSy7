## 🌍 About Me

Frontend developer from Coyah, Guinea. Self-taught since 2022.

I started with YouTube tutorials, Grafikart, Traversy Media, Kevin Powell, learning HTML, then CSS, then spending about eight months getting comfortable with JavaScript. No bootcamp, no CS degree. Just docs, replayed videos, and a lot of broken code that eventually stopped being broken.

Living in Guinea makes some things harder than they should be. Power cuts mid-session, internet that drops during downloads, no credit card for paid courses. I found ways around most of it. Some of the best things I've learned came from courses I couldn't afford to buy.

In 2024, I entered Enzo Ustariz's web dev contest, not for the first time but for the first time seriously. Two weeks building a tourism site for Conakry. Came third. That felt like proof the approach was working.

A few months later: a remote frontend internship at DevelopersHub Corporation. Six weeks, real tasks, real code review, exceptional distinction certificate on exit.

Currently working with React, Next.js, TypeScript, and GSAP. Learning Node.js. Open to remote work.
